# Yamtrack for YunoHost

> *This package allows you to install Yamtrack quickly and simply on a YunoHost server.*

## Overview

Yamtrack is a self-hosted media tracker for movies, TV shows, anime, manga, games, books and more.

**Shipped version:** 0.25.0~ynh1

**Demo:** <https://yamtrack.fuzzygrim.com>

## Documentation and resources

- Upstream app code repository: <https://github.com/FuzzyGrim/Yamtrack>
- Report a bug: <https://github.com/gmanaud/yamtrack_ynh/issues>
